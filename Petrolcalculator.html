Petrol Cost Calculator

This is a simple calculator which calculates the cost of buying Petrol. It is not yet possible to change the litre price of petrol, this is coming in V2!

How many litres do you want to buy?

The total cost will appear here

Explanation:

There's a simple form which takes an input the number of litres petrol-litres and on click runs the function go().

This is the form code: <input> id="petrol-litres" type="text"> <button onclick="go()">Calculate total cost</button>

When the program is started, it starts half way down at the "go" function.

  1. The "go" function is concerned with the presentation of the results.
  2. The first thing the "go" function does is speficy that the variable "noLitres" is the petrol-litres value which was entered into the form.
  3. Next the "go" function specifies that the variable "cost" is equal to the function "petrolCost with the parameter "noLitres". This prompts the program to head back to the top of the script and run the "petrolCost" function.
  4. The "petrolCost" function speficies the amountPayable is equal to the calculation (quantity * price). This is the calculation bit.
  5. The next line returns the "amountPayable" variable to the last line of the code (within the "go" function") and displays the result.
  6. The final line of code uses the "getElementByID" function to replace the contents of a div on the page with the result.

This is the code itself: 

function petrolCost(petrolQuantity) //calculation//
{        
    var amountPayable = petrolQuantity * 1.3;
    return amountPayable; //return is the answer from the function to the caller//
}
            
function go() //presentation of results/// 
{
    var noLitres = document.getElementById("petrol-litres").value;
    var cost = petrolCost(noLitres);  //this is the 'caller'//
    document.getElementById("cost-display").innerHTML = 'The cost is: £' + cost    
}
